## Setup IntelliJ IDEA

Windows:
```
IntelliJ IDEA -> File -> Settings -> Plugins
```

macOS:
```
IntelliJ IDEA -> Preferences -> Plugins.
```

Install `Cucumber for Java` plugins and restart.
